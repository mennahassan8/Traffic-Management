# Traffic-Management (Data Structures)

# For Modern smart cities require intelligent traffic management to balance efficiency, safety, and fairness. This project simulates a sophisticated Traffic Control Center (TCC) for a smart city. It is an object-oriented, event-driven C++ simulation designed to allocate green light slots fairly to different vehicle types while giving strict priority to emergency vehicles. The system dynamically handles multiple intersections, interconnected roads, and unexpected occurrences like accidents or road closures.
